# 📊 Customer Churn Prediction

A Machine Learning and Power BI project that predicts customer churn and identifies the major factors responsible for customers leaving a telecom company.

---

## 📌 Project Overview

Customer churn prediction helps businesses identify customers who are likely to leave their services.

This project combines Machine Learning and Power BI to generate business insights and predictive analytics.

---

## 🎯 Objectives

- Clean and preprocess customer data
- Perform Exploratory Data Analysis (EDA)
- Train Machine Learning models
- Predict customer churn
- Identify important churn factors
- Build an interactive Power BI dashboard

---

## 📂 Dataset

Telco Customer Churn Dataset

Customers: **7043**

Target Variable:
- Churn (Yes/No)
